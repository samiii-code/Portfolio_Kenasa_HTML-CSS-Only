## 🎯 Project Goal

The goal of this portfolio is to create a **modern, visually stunning, and interactive website** using only **HTML and CSS**, without relying on JavaScript or frameworks.

It focuses on:
- Clean and professional UI/UX
- Smooth animations using pure CSS
- Responsive design for all devices
- Creative and futuristic visual effects
